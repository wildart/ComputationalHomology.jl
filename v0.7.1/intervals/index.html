Intervals

Any interval type need to implement following interface:

ComputationalHomology.AbstractInterval — Type

Abstract interval type

source
ComputationalHomology.birth — Function
birth(i::AbstractInterval)

Return a birth value of the interval i.

source
ComputationalHomology.death — Function
death(i::AbstractInterval)

Return a death value of the interval i.

source

The persistence diagram is defined as

PersistenceDiagram{T} = AbstractVector{<:AbstractInterval{T}}
